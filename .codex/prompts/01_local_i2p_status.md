# Prompt 01: Local I2P Status

Design the first runtime PR for measured local I2P readiness.

Requirements:

- inspect existing docs first
- choose stack and test strategy before code
- use localhost-bound router integration
- never expose router/admin/control ports publicly
- report measured states only
- add tests for reachable, unreachable, and misconfigured states

Real-time video and guaranteed anonymity are not promised.
